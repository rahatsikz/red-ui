## RED UI

Red UI is a React.js component builder library built with Next.js, TypeScript, and Tailwind CSS. It offers developers a wide range of components to use in their projects, providing flexibility and ease of use

## Live Site

[https://red-ui.vercel.app](https://red-ui.vercel.app)

## Used Technology

- Next.js (version 14.1)
- TypeScript
- Tailwind CSS (version 3.3)

## Feature

- Generate clean and responsive UI code with Tailwind CSS for faster development
- Preview components instantly with real-time rendering and interactive editing
- Export ready components as reusable modules for easy integration across projects
- Build lightweight, fast-loading UI components with minimal dependencies
- Create custom, user-friendly interfaces without the bloat of unnecessary libraries
- Enjoy rapid development cycles with Tailwind's utility-first approach and React's component-based architecture
- Seamless performance by leveraging the power of progressive web apps
